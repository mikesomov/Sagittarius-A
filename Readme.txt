我們擁有 136 億年歷史的星系中的一切都圍繞著人馬座 A* 運行，包括我們距離我們 26,000 光年遠的太陽系 。

眾所周知，黑洞很難被發現，通常只能透過它們對環境的影響來推論。這是因為它們不僅不會發光，而且還將光子捕捉在稱為事件視界的邊界後面，這使得直接在可見光中研究它們幾乎是不可能的。

從地球上觀測人馬座 A* 變得更加困難，因為它被一層厚厚的塵埃所籠罩。

幸運的是，天文學家已經發展出其他方法來深入了解人馬座 A*。例如，中心天體的質量及其半徑可以透過觀察其對繞其運行的物體的引力影響來確定。

為了觀測人馬座 A*，天文學家監測了恆星 S2 ，該恆星繞人馬座 A* 運行，距離為 110 億英里（180 億公里），速度為 1710 萬英里/小時（11.4 公里/小時） 。這顆恆星還有一個高度橢圓的 16 年軌道 。
